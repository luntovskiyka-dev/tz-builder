## 🎯 Функциональность

### 1. Просмотр текущей подписки
- **Название тарифа** (Starter, Project, Pro, Studio)
- **Статус** с цветным индикатором:
  - 🟢 `active` — активна
  - 🟡 `trialing` — пробный период
  - 🟠 `past_due` — просрочена
  - 🔴 `canceled` / `expired` — отменена/истекла
- **Даты**: триал до, следующее списание, тип доступа
- **Предупреждение** для триал-периода

### 2. Дашборд использования AI
- **Дневной лимит**: прогресс-бар + остаток
- **Месячный лимит**: прогресс-бар + остаток
- **Цветовая индикация**:
  - 🟢 < 70% использования
  - 🟡 70-90%
  - 🔴 > 90%
- **Предупреждение** при низком остатке
- **Поддержка безлимитных планов** (лимит = 0)

### 3. История платежей
- Таблица последних 50 платежей
- **Колонки**: название, дата, сумма, статус
- **Статусы** с цветными бейджами:
  - Ожидание, Успешно, Ошибка, Отменен, Возврат
- **Загрузка** с состоянием и обработкой ошибок

### 4. Смена тарифа
- Компактные карточки 4 тарифов
- **Текущий тариф** выделен рамкой и бейджем
- **Бейджи**: "Популярный" (Pro), "В разработке" (Studio)
- **Логика**:
  - downgrade на Starter — мгновенно, без оплаты
  - upgrade с триалом — мгновенная активация
  - upgrade без триала — редирект на оплату
- **Предупреждение** о последствиях

### 5. Отмена подписки
- Модальное окно подтверждения
- **Список последствий**:
  - Доступ прекратится в конце периода
  - Проекты сохранятся
  - Можно возобновить позже
- **Состояние загрузки** с спиннером
- **Перезагрузка** после успешной отмены

## 📝 Заметки

### Безлимитные планы
- `daily_limit = 0` или `monthly_limit = 0` означает **безлимит**
- Прогресс-бар показывает минимальное использование
- Цвет всегда зеленый

### Смена тарифа
- **Downgrade** на Starter — мгновенно
- **Upgrade** с триалом — мгновенная активация
- **Upgrade** без триала — редирект на ЮKassa

### Отмена подписки
- Пока только обновляет статус в БД
