## Текущий активный контекст

На данный момент в проекте:

- Создан базовый макет дашборда с тремя колонками.
- Левая панель (260px, серая) показывает библиотеку блоков: Hero, Текст+Изображение, Карточки товаров, Форма, Отзывы, FAQ, Footer.
- Центральная область — канвас с светло-серым фоном и мелкой сеткой, поддерживает drag & drop блоков с помощью **dnd-kit**.
- Блоки на канвасе представлены компонентом `BlockOnCanvas`, они сортируемые (можно менять порядок перетаскиванием).
- Есть состояние для списка блоков на канвасе, включая сохранение их типа и настроек.
- Реализован инспектор свойств в правой панели (280px, серая): при клике по блоку он подсвечивается, а справа отображаются поля настройки (разные для Hero, Карточек товаров, Текст+Изображение, Формы, Отзывов, FAQ и Footer).
- Используются компоненты **shadcn/ui** (`Input`, `Checkbox`, `RadioGroup`, `Textarea`, `Label`) для формы инспектора.
